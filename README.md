# plan.py — Генератор плана тренировок

Скрипт читает список упражнений и текущую неделю, случайно распределяет упражнения по трём тренировкам и создаёт HTML-файл, который можно открыть в любом браузере.

---

## Структура папок

```
любая_папка/
├── plan.py
└── Exercises/
    ├── exercises.txt      # список упражнений и приростов
    ├── week.txt           # одно число: номер текущей недели (0, 1, 2, …)
    ├── 1.png              # фотографии (имена — целые числа)
    ├── 2.png
    ├── song1.mp3          # музыка для плейлиста
    └── song2.mp3
```

---

## Требования

- Python 3.9 или новее
- Сторонних библиотек не нужно — только стандартная библиотека

---

## Запуск

Откройте терминал в папке, где лежит `plan.py`, и выполните:

```bash
python plan.py
```

После этого рядом с `plan.py` появится файл `workout_plan.html`. Откройте его двойным кликом в браузере.

---

## Формат exercises.txt

Файл состоит из двух частей, разделённых строкой `Увеличения через каждый выходной:`.

**Часть 1 — упражнения** (15 штук, по 3 в каждой из 5 секций, секции разделены пустой строкой):

```
1. Название упражнения МИН-МАКС единица
```

Пример:
```
1. Джампинг Джек 20-60 секунд
2. Бег на месте с подъёмом коленей 20-60 секунд
3. Горизонтальные альпинисты 15-40 повторений на сторону
```

**Часть 2 — приросты** (по одному на каждое из 15 упражнений):

```
1. +4 секунды
2. +4 секунды
...
```

---

## Формат week.txt

Файл содержит одно целое число — номер текущей недели:

```
3
```

Число повторений или секунд для каждого упражнения вычисляется по формуле:

```
N = min(МИН + ПРИРОСТ × НЕДЕЛЯ, МАКС)
```

Чтобы перейти к следующей неделе, увеличьте число в `week.txt` на 1 и запустите скрипт заново.

---

## Что делает HTML-файл

| Элемент | Поведение |
|---|---|
| **Фото** | При каждом открытии файла случайно выбирается одно из `.png`-изображений в папке Exercises |
| **Плейлист** | Автоматически воспроизводит случайный трек из всех `.mp3` в папке Exercises; треки переключаются по окончании |
| **Тренировки** | Три карточки (А, Б, В) — в каждой по 5 упражнений, по одному из каждой секции; упражнения не повторяются между тренировками |
| **Объём** | Рядом с каждым упражнением указано рассчитанное значение N и единица измерения |

---

## Примечания

- Если в папке нет `.mp3`-файлов, плейлист остаётся пустым и плеер не мешает работе.
- Если в папке нет `.png`-файлов, блок с фото просто не отображается.
- Каждый запуск `plan.py` перегенерирует распределение упражнений — тренировки А, Б, В каждый раз новые.
